# Agency Digital – Recruitment Pipeline

A React + Vite + Tailwind implementation of the Recruitment Pipeline dashboard
(Manulife Agency Digital), converted from a Google Stitch HTML export.

## Stack

- React 18
- Vite 5
- Tailwind CSS 3 (custom Material 3-inspired token theme)

## Getting started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Structure

```
src/
  components/   # SideNavBar, TopNavBar, PipelineHeader, KanbanColumn, CandidateCard, AICopilotDrawer
  data/         # candidates.js - kanban column + candidate data
  App.jsx
  main.jsx
  index.css
```
